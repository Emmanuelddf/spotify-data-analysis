# Spotify Data Analysis

Analyze Spotify tracks, artists, and albums to uncover trends, popularity, and patterns.  

## Highlights
- Cleaned and processed data using Python  
- Explored trends in track popularity, explicit content, and release patterns  
- Interactive dashboards built in Power BI  

## Tech
Python | Pandas | NumPy | Power BI  

## Usage
Clone the repo and explore the dataset or dashboards for insights.
