
# Spotify Data Analysis – Project Summary

This folder contains the analytical outputs for the Spotify Data Analysis project, including:

- **Insights.md** — Key findings and insights from the dataset
- **EDA results** from Jupyter Notebook
- Documentation files to support the Power BI dashboard and Python analysis

## Files in This Folder

- **Insights.md** – Detailed breakdown of findings such as duration trends, popularity distribution, decade performance, and artist/album insights.
- **README.md** – Documentation for the analysis section of the project.

## Project Overview

The full project analyzes Spotify track data using:
- **Python** (for cleaning, transformation, and exploratory data analysis)
- **Power BI** (for visualization and dashboard creation)

Outputs are organized for use in both my **portfolio** and **GitHub repository**.
